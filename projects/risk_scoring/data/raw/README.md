# Raw Data: Lending Club (OpenIntro)

This project uses the OpenIntro "loans_full_schema" dataset, which is an open
sample derived from Lending Club loan data.

Source page:
https://www.openintro.org/data/index.php?data=loans_full_schema

Direct CSV:
https://www.openintro.org/data/csv/loans_full_schema.csv

Place the CSV in this folder as:
`data/raw/loans_full_schema.csv`
